2023.09.11
- 전역변수 next로 인한 버그 수정
* 개발자 코멘트
개인적으로 전역변수 사용을 좋아하지 않습니다.
그 이유는 코드가 길거나 작성한지 오래되었는데, 동시에 전역변수를 사용했다면,
변수 적용벙위에 있어 혼란이 있을 수 있기 때문입니다.
역시 이번에도 전역변수 이슈로 버그가 생겼습니다. 버그 내용은 정답을 맞췄을 때,
다음 문제로 넘어가지 않고 0번 문제가 나오는 것이었습니다.
다행히 현재는 버그가 수정된 상태입니다.

- 정답 입력 시 이미지 전체 공개 기능 추가
* 개발자 코멘트
정답 입력 시 이미지 전체를 공개하는 기능을 추가할 경우,
정답이 무엇인지 모두가 쉽게 확인하는 효과가 기대됩니다.
이에 따라, 정답 입력 시 이미지 전체를 공개하는 기능을 추가했습니다.

- 랜덤 순서 로직 삭제 및 고정 순서 로직 추가
* 개발자 코멘트
랜덤으로 문제가 출제되는 세팅은 출제자가 정답을 쉽게 확인하지 못할 수 있다는 피드백이 있었습니다.
이에 따라, 문제 순서가 고정되게 나오도록 변경했습니다.

- cmd 나오지 않게 패키징
* 개발자 코멘트

이에 이 기능을 제거하는 옵션을 넣어 패키징 하였습니다.
